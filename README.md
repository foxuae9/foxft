# TikTok Mafia

A Next.js web application for managing TikTok Mafia tournaments and players.

## Features
- Player management
- Tournament tracking
- Admin dashboard
- Real-time updates

## Setup
1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
